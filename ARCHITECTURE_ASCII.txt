║                                                                              ║
║ 1️⃣  CREATE BOOKING (with pricing):                                          ║
║    User Form → BookingController.save() → PricingStrategyService →         ║
║    DiscountStrategyFactory → DiscountStrategy → PriceCalculator →          ║
║    BookingRepository.save() → DB INSERT → HTMX refresh                     ║
║                                                                              ║
║ 2️⃣  PAYMENT PROCESSING:                                                     ║
║    User clicks Pay → BookingController.pay() → Booking.pay() →             ║
║    current_status.pay() [STATE PATTERN] → sync status → save() →           ║
║    BookingStatusChangedEvent → EmailNotificationListener → send email      ║
║                                                                              ║
║ 3️⃣  CANCEL BOOKING:                                                         ║
║    User clicks Cancel → BookingController.cancel() → Booking.cancel() →    ║
║    current_status.cancel() → CancelledState → save() → event publish       ║
║                                                                              ║
║ 4️⃣  QUERY TOUR DATA:                                                        ║
║    GET /tours → TourController.list() → TourRepository.findAll() →         ║
║    buildOccupiedByTourId() [calc occupancy] → load dropdowns →             ║
║    render tours-list.html with HTMX                                        ║
